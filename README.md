# 🤖 Aternos Auto-AFK Bot Dashboard v2.0

Stop worrying about your Aternos server shutting down every time you go to sleep. This is a web-based dashboard designed to keep your Minecraft server alive 24/7 using automated bots.

## ✨ Why this bot?
*   **Dual Support:** Works for both **Minecraft Bedrock** and **Java Edition**.
*   **Paper/Spigot Compatible:** Connects perfectly to Paper servers (even via GeyserMC for Bedrock users).
*   **Auto-Restart:** If the host restarts, the bot automatically reads your `settings.json` and jumps back into the game.
*   **Web Dashboard:** Real-time logs and controls right in your browser.
